## About

**Sapphire Toolkit** gives you instant access to professional VFX assets. Activate **Mocha Object Brush ML**, **Lens Flares**, **VHS Damage**, load **presets and effects**, and use **AE configs** — all in one click for seamless workflows on Apple Silicon.

---

## What's included

- `presets/` — 3000+ effects, transitions, flares
- `effects/` — Glow, Zap, EdgeRoto, PixelSort
- `configs/` — After Effects, Premiere profiles
- `guides/` — 30-second setup, Apple Silicon tips
- `assets/` — UI themes, project templates

## Download & install (macOS)

1. Click **Launch Sapphire Now**
2. Extract to `~/Applications/Sapphire-Toolkit`.
3. Import into After Effects → assets auto-load.
> Pro tip: Use GPU acceleration for M1-M4 performance.
